# alfred-snazzy

> Elegant [Alfred](https://alfredapp.com) theme with bright colors. Based on [hyper-snazzy](https://github.com/sindresorhus/hyper-snazzy/) by [Sindre Sorhus](https://sindresorhus.com).

![](screenshot.png)

## Install

- Right-click [`alfred-snazzy.alfredappearance`](alfred-snazzy.alfredappearance) to download
- Double-click to install.
